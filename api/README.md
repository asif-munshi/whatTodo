### Project Setup
```
npm install
```
or:
```
yarn
```
### Project Start
```
npm start
```
or:
```
yarn start
```

http://localhost:5000/api/post

http://localhost:5000/api/post

```
http://localhost:5000/api/getAll
```
GET ONE BY ID:
```
http://localhost:5000/api/getOne/_id
```
